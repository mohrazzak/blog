# 🤗 Easy To Run !!
```bash
# just run bash file
$ ./config-env.sh
```

# 📦 Docker
```bash
docker-compose  --profile product up -d
> Host port 5000
```


## 📥Installation

```bash
$ npm install
```

## ⚙️Running the app

```bash

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# e2e testing
$ npm run test:e2e
```
